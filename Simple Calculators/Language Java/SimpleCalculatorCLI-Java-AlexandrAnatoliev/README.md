 # Simple Calculator 

#### Compile Java code
``` 
javac -d bin/main/ src/main/java/*.java
```
#### Using program
* Move to directory
```
cd bin/main/
```
* Run program
```
java Main 2 + 3
2 + 3 = 5
java Main 6 - 3
6 - 3 = 3
``` 
* Warning!!! To multiplication use 'x' operator instead '*'
```
java Main 2 x 4
2 x 4 = 8
```
