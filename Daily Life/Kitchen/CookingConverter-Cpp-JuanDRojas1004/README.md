# Cooking Measurement Converter (C++)

A C++ program that converts between common kitchen measurements.  
Useful for cooking and baking!

---

## Supported Conversions

- Cups ↔ Tablespoons  
- Tablespoons ↔ Teaspoons  
- Cups ↔ Milliliters  
- Tablespoons ↔ Milliliters  

Uses standard reference conversions:
- 1 Cup = 16 Tablespoons  
- 1 Tablespoon = 3 Teaspoons  
- 1 Metric Cup = 250 ml  
- 1 Tablespoon = 15 ml
